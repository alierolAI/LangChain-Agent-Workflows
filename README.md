# LangChain Agent Workflows

Practice projects and experiments focused on LLM orchestration, LangChain, LangGraph, Retrieval-Augmented Generation (RAG), and AI agent workflows.

## Overview

This repository contains hands-on projects developed while learning modern AI application development using Large Language Models (LLMs).

The main goal of this repository is to explore how advanced AI systems can be built using orchestration frameworks such as LangChain and LangGraph.

Topics include:
- Prompt engineering
- AI agents
- Workflow orchestration
- Memory systems
- Retrieval-Augmented Generation (RAG)
- Vector databases
- LLM integrations
- Message history management
